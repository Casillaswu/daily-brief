§02Strategic Read · 5Q

策略解读 · 5 个问题

Q1
市场空间

PAGCOR 5/15 截止后市场结构出现两个分裂：

· 合规侧：1.5% turnover / 15% net loss 上限正式执行、「先前批准、非合规」的旧促销方案必须下架。EGLD 收件箱预计塞满 EG Form No. 28 申请。
· 营销侧：9 家应对策略主流不是「砍数字」而是「换字眼框架」——把 现金回扣 改挂迎宾 / 首存 / 续充奖励标签、归类边界等待 PAGCOR 后续解释（9 家详情见 §04 卡 1）。

PAGCOR 这条规如果不再补充细则界定「持续型 vs one-time」边界，1.5% 上限的实际管控力可能被「字眼定位」稀释。这是接下来 4-8 周的关键观察点。

出处：IAG 5/10 · GGRAsia · 9 家官网偵察 5/18
Q2
玩家关心

本周玩家三大关注点（依社群高频话题排序）：

· 「King / Queen 上的钱怎么办」——双站维护进入第 4 周、World Platinum 0 公告，玩家资金与提现疑虑积累。本周话题热度盖过 PAGCOR 法规。
· 「我的 现金回扣 还在吗」——5/15 后原本拿 CasinoPlus 2.6% / Bet88 5% 周打的玩家在群里问运营商。各家客服回应不一。
· 「BingoPlus Night 5/19 怎么参加」——转入活动期、Vice Ganda + Kim Chiu 阵容拉抬话题热度。

出处：PH 玩家社群高频话题 + king.ph / queenking.ph 偵察 5/18
Q3
缺口

5/15 发薪日 → 5/24 周末 → 6/12 独立日，三档期连续，9 家本周仍 0 档期素材。菲律宾父亲节 6/15 进入暖场期，0 家有动作。

「迎宾字眼定位下的真实价值」——玩家越来越意识到 60% / 30% / 100% 数字是首存一次性、月度有效价值远低于过去的 持续型返水。若运营商能向玩家说明每月真实可领回流、可拿到「合规叙事 + 玩家信任」双轴差异化（执行方式 hedge、详 §06 机会 3）。

King + Queen 第 4 周事故，至今 0 家本地媒体跟进。新闻空白窗如果延续到第 5 周，将明显提高玩家迁移与信任修复成本。

出处：本周 9 家 /promotions 偵察 5/18 + PH 媒体 King.ph 关键词 0 报导
Q4
价档

PAGCOR 5/15 后各家「字眼标示」vs「实际可领」对照（详细机制见 §04 卡 1 / 4 / 5）：

· OKBet「Top-Up 60% Rebate」= 首存迎宾、限 48 小时
· BingoPlus「30% 现金回扣」= 首存 ₱3K → ₱900 一次性
· Bet88「5% 每周 + 100% 存款奖励」= 续充 + 迎宾分层
· CasinoPlus 2.6% 现金回扣 仍挂（归属待解释）
· PlayTime 已下架、改挂 ₱7,730 单档

判断：「返水数字大小」差异化时代结束、护城河转向 IP 锁定、本土文化内容、单游戏 SKU。

出处：各家官网 /promotions 偵察 5/18 · iGB 5/12 · RULES.md §3.2
Q5
差异化

W21 各家路线总结：DigiPlus 三品牌矩阵周封顶（BingoPlus Night）/ PlayTime MUPH 余波 / CasinoPlus 本土 IP 维持 / OKBet 双轨字眼 / Bet88 多档分拆 / King + Queen 流失承接窗口。

结论：下半年护城河在「内容稀缺性 + IP 锁定 + 合规叙事透明度」，单靠返水数字的运营商被结构性削，2026 H2 利润排序会洗牌。

出处：本周 9 家公开动作汇总 + DigiPlus 5/12-5/16 集团 PR 接力 + PAGCOR 5/15 deadline 交叉
