§02Strategic Read · 5Q

战略解读 · 五问（中性陈述）

Q1
市场空间

市场在缩。PAGCOR 预估 2026 全年博彩收入 ₱320-350 亿、比 2025 的 ₱396.14 亿少最多 19%；第一季 GGR 年减 15.87% 至 ₱87.6 亿，其中线上电子博彩年减 22.43%。PAGCOR 主席 Tengco 把线上衰退主要归到中东危机——油价 / 运输成本升高、压到中低收入族群的花费（其余因素与细节见 §04 卡 3）。监管持续在收：B2B 8/1 下架、KYC 收紧、广告只禁黄金时段且全日禁播研议中、24 小时戒赌专线（后三项来自 2 月参议院听证、属持续型背景）。供给端本周最大动作，是 GameZone 帮 LaVie 实体赌场做线上平台。

一句话：市场缩、获客难，单纯拼促销数字的边际效益在掉；龙头把战场拉到「规模 + 生态」。若广告全日禁播落地，九家靠电视 / KOL 的打法都得重做。

出处：Yogonet 6/9 · tribune 第一季 -16% · PNA 2 月听证
Q2
玩家关心

玩家在意的，跟产业大新闻基本不交集。本周三件大事（LaVie 合作 / 世界杯开赛 / B2B 倒数）玩家端没什么讨论。玩家真正在吵的是三件：一是钱出不来、存不进去（存款常卡——KYC 前置 + 支付通道限制，提款拖几天到几周的抱怨一直有，法律咨询站把「提款迟迟不到」列为选平台红旗）；二是赌债与成瘾（警方网络犯罪部门光 2026 第一季就收到约 1,891 件非法线上博彩检举，媒体估全菲超过 3,000 万人在线上博彩，教会与公卫界当社会危机在讲）；三是冒名诈骗（品牌名被拿去做钓鱼连结，老问题）。

一句话：玩家要的是钱进得去、出得来、不被骗，外加戒赌资源；不是再多一个促销数字。客户 X 想打中玩家，差异点在「存款顺、提款快、KYC 透明、防诈」。

出处：网络犯罪部门 Q1 检举数 · gulfnews 成瘾危机 · 提款问题法律指引（长期结构性议题、非本周事件特定）
Q3
缺口白旗

入金路径教育：玩家存款本来就常卡（KYC + 支付通道限制），但九家本周没一家出「替代存款方式 / 怎么存进去」的教学型内容。真空白。

父亲节档期：6/21 父亲节（本周日）+ 6/15 发薪日，九家到本周还没动父亲节素材（trigger 监听 6/8-6/15 逐日确认）。

B2B 合规对外沟通：8/1 供应商下架在即，九家没一家公开「我家供应商认证到位了没」。

注：上周列的「世界杯促销空白」本周已不成立——四家都推出了世界杯促销（见 §04 卡 2），从缺口移除。

出处：trigger 监听 6/8-6/15 + 6/15 实地看页
Q4
价档结构

6/15 实地所见：

返利回馈线：CasinoPlus（1.5% 返利 + 1.5% 回归）、OKBet（主站 1.5% 日返）、Bet88 / PlayTime（1.5% 持续型）这几家非 DigiPlus 集团的运营商，主推的返利大多落在 1.5% 上下、彼此越来越像。世界杯加码：四家把世界杯当主打——OKBet FIFA 2026 老虎机奖池 ₱15,888,888（最大）、Bet88 ₱4,900,000 奖池 + 25% 回馈、CasinoPlus World Cup Challenge ₱478,800、ArenaPlus World Cup Daily Bonus。迎宾档：Bet88 100% 首存 ₱8,888、PlayTime ₱8,080、BingoPlus 30% 一次性。

一句话：这周价档本身没什么变化，各家给的优惠没有明显变多或变少；真正动起来的是世界杯专档，一周内从一家变四家。接下来看玩家存款这道摩擦，会不会逼大家把重点从「奖金多大」改成「钱存得进去」。

出处：6/15 实地看页（CasinoPlus / OKBet / Bet88 / ArenaPlus）
Q5
差异化结论

DigiPlus 系靠 LaVie 合作把战略维度再拉一层（GameZone 从自营站变成「帮别人做平台」）；世界杯档四家齐发、谁都没缺席；非集团的几家运营商（CasinoPlus / OKBet / Bet88 / PlayTime）返利都靠到 1.5% 上下；GameZone 因 LaVie 合作从静默转热；OKADA PLAY 仍低位。

结论：竞争题目又往上挪了一格。上周是「谁的规模与合规更可信」，这周 DigiPlus 再加一手「谁能把实体赌场、帮别人做平台、自营线上串成一个生态」。对中型独立运营商，多了一条新压力：龙头开始当平台供应方，可能反过来变成你的上游。世界杯这种短窗口也证明——空档关得很快（一周内从一家变四家），靠「别人没做」抢辨识度的机会越来越短命。

出处：本周九家公开动作汇总 + 6/15 实地看页 + DigiPlus × LaVie 合作多源

§06Implications · 威胁 / 机会

影响推演（威胁 / 机会）

进场参数参考（皆为内部估算）
  • 龙头生态化的追赶成本：非上市进场者若想要对等的「规模 + 生态」叙事，光治理框架 + 第三方背书就估 ₱10M-₱30M + 6-12 个月。
  • 进场量要下修：市场在缩（GGR 预估年减最多 19%）+ 固定月费地板（MGF），首年获客目标建议下修 15-25%。
  • 入金成功率是要盯的指标：PH 玩家存款本来就有摩擦（KYC + 支付通道）；进场模型把「入金成功率 × 首存转化」纳入核心、不只看 CPA。
  • 世界杯档期：6/11-7/19 约 38 天进行中；Altenar 等级 fully-managed 体育博彩平台授权估 ₱8M-₱15M / 年。
威胁（5 条）Threats
  • 龙头开始当平台供应方、可能从对手变上游。GameZone 帮 LaVie 做线上平台，DigiPlus 从「自营线上」跨到「实体 + 帮别人做平台」。中型独立运营商与客户 X 若没有对等的规模 / 生态，声量与议价会结构性落后；要是去租龙头的平台，等于把玩家数据交给最大对手。不动的代价（内部估算）：自建生态 vs 依附龙头平台两难，追赶窗口随龙头生态变完整而收窄。
  • 市场规模缩减、固定月费却有地板。2026 GGR 预估年减最多 19%、线上第一季 -22.43%；PAGCOR 最低保证费（MGF）第一批 6/1 起跑（线上 e-casino 月最低 ₱9M、体育 ₱3M，不管营收都要缴）。市场缩、成本有地板，进场即逆水。不动的代价（内部估算）：误判市场还在长、首年目标偏高 15-25%，又没把 MGF 算进 burn，ROI 下修。
  • 入金摩擦是结构性问题。PH 玩家存款本来就常卡（KYC 前置 + 支付通道限制），线上第一季 -22.43%；只盯 CPA 不盯入金成功率的获客模型会失真。不动的代价（内部估算）：进场后入金成功率低，CPA 看似达标但首存转化崩，有效获客成本被低估 30-50%。
  • 8/1 供应商下架在即，供应链可能断。九家依赖的供应商若有几家 5/31 没递件 / 7/31 没过审，8 月初可能 1-3 家短暂中断。不动的代价（内部估算）：进场后替换备援 ~₱2M-₱5M，上线延后 4-8 周。
  • 广告可能全日禁播，SOV 打法有结构性风险。现在只禁黄金时段；若扩到全日 + 数位，九家靠电视 / KOL / 社群的打法都得重做。不动的代价（内部估算）：渠道重构，获客成本短期 +20-40%。
机会（4 条）Opportunities
  • 入金教育没人做（内部估算成本 ~₱200K-₱500K / 4 周）。PH 玩家存款本来就多一道手续（KYC + 支付通道限制），九家本周没一家出「怎么存 / 替代金流」教学。客户 X 可做「入金教学 + 替代金流引导」，把支付摩擦转成玩家信任素材。反向提醒：替代金流引导一律法务先看，不碰「保证 / 稳赚」字眼，不引导绕过合规通道。
  • 父亲节档期没人插旗（内部估算成本 ~₱500K-₱1M / 4 周）。6/21 父亲节、九家本周零动作。客户 X 可做品牌情感沟通（不促销、不碰家庭 / 儿童意象），纯打「父亲」情绪 + 责任博彩讯息。反向提醒：博彩碰节庆的红线是「触及未成年与家庭」，文案与渠道一律法务先看。
  • B2B 合规叙事没人讲（内部估算成本 ~₱200K / 4 周）。8/1 下架在即，九家没一家公开自家供应商认证状态。客户 X 可做「全栈认证叙事 + 供应商状态公开页」，把 8/1 压力转成差异化素材。反向提醒：若「合规叙事」被当成行业义务，差异化窗口很快消失。
  • 世界杯之后看「执行 + 在地化」（内部估算成本视档期定）。四家世界杯专档齐发、「有没有做」已不是差异点；接下来差在玩法、奖池规模、在地化叙事。客户 X 若有体育线，差异要押在执行细节、不是档期本身。反向提醒：四家已先动，进场前先实地盘对手的实际玩法、别对着旧情报做。
